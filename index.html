Image Cropper

By Gabriel Angel













Tutorial

Click the upload button to select any number of JPG or PNG files. After pressing the select button, these files will be loaded into the program for cropping. Next, manually enter the amount of pixles you would like shaved off from each side of the image, and... voila! The Image Cropper will automagically crop all of the selected images simultaneously with the same dimensions. No more needless, repetitive cropping! But don't forget to select the radio button for either overwiting existing files, or saving new ones.

This may not be a useful tool for you, but it is for me, and that is why I made it. If you want user-friendly, drag-n-drop cropping, there are a plethora of other free online options for you to find with any search engine. This tool is specifically designed to crop many images at once to the exact same dimensions.
